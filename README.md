# Student Management System

A modern student management web app built using React and Material UI.

## Features
- Add Student
- Edit Student
- Search Students
- Filter Students
- Export to Excel
- Mobile Responsive UI

## Tech Stack
React
Material UI
Vite
SheetJS (Excel Export)
